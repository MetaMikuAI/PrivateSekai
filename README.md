# Private Sekai

## 项目简介

本项目用于模拟游戏 **初音未来 -歌姬计划- （初音ミク -プロジェクト ディーヴァ-，Hatsune Miku: Project DIVA）** 游戏测试服务端，供个人技术研究、协议分析、逆向学习等用途。所有功能仅限于本地测试，严禁用于任何形式的公开传播、开服或商业用途。

## 游戏介绍


>
> ​	《初音未来 -歌姬计划-》系列（繁体中文：初音未来 -名伶计划-系列，日文：初音ミク -Project DIVA- シリーズ）是由SEGA主导开发的音乐节奏游戏系列。游戏初登场于PSP平台，后来在PSV、PS3、街机、PS4、Switch、PC等平台不断推出新作。系列后来扩展为SEGA feat. HATSUNE MIKU Project。
>
> ​	此系列首款游戏《初音未来 -歌姬计划-》发布于2009年7月2日，由DINGO和SEGA共同开发，几年后DINGO破产，游戏变为由世嘉独立主导开发。
>
> ​	游戏名称中“DIVA”是歌剧中的女主唱的意思，并且游戏以Crypton Future Media株式会社旗下的虚拟歌手初音未来命名。该游戏系列的绝大多数曲目均由VOCALOID虚拟歌手初音未来、镜音铃·连、巡音流歌、MEIKO和KAITO演唱。
>
